Schlagzeilen

Anzeige „Nur Subjekte" zeigt eine Zeile je erkanntem Subjekt — Ticker, aber auch Personen/Indizes (z. B. Trump, S&P 500). „Subjekte + Threads" zusätzlich die Thread-Themen.
Bilder mitanalysieren Wertet Bilder aus Threads und Kommentaren per Vision-Modell aus und lässt sie in Schlagzeilen einfließen. Aus = schnellere, reine Text-Schlagzeilen ohne Bildanalyse.

Sprache

Anzeigesprache Wird nach einem Neustart vollständig übernommen.

Updates

Automatisch aktualisieren Aktualisiert beim Start automatisch. Wenn aus, meldet ein Hinweis in der Titelleiste neue Versionen.

Daten

Daten löschen Setzt das Terminal komplett zurück: Threads, Cluster, Schlagzeilen und das Archiv. Danach füllt sich die Wire vom nächsten Scan neu. Höchstens alle 10 Minuten.
