# Decision/Research — Daten-Quellen (Preis, Auflösung, News) + Yahoo-Unabhängigkeit

**Stand:** Yahoo wahrscheinlich **dauerhaft IP-gebannt** (wie der Reddit-`.json`-Pfad). Wir machen die Pipeline schrittweise Yahoo-unabhängig. Branch `feat/editorial-lab-per-subject`.

## Zwei getrennte Rollen (nicht vermischen)
1. **Auflösung** (Name → Ticker): kann **lokal + anonym** werden.
2. **Live-Preis** (Ticker → Kurs/Move): braucht eine Quote-Quelle.
(News ist eine dritte, separate Ebene — siehe unten.)

## Auflösung — Yahoo-frei via lokalem Korpus + Tier 2
- **SEC `company_tickers.json`** (sec.gov, anonym, kein Key) = komplette US-Ticker↔Name-Liste. DE/Xetra bräuchte eine zweite statische Liste.
- **Tier 2** (Embedding-Ranker, gebaut, commit `19de14d`) rankt Kandidaten semantisch → Name→Ticker **offline**, kein Rate-Limit, kein Key. Heute rankt Tier 2 Yahoo-Kandidaten; designt, um denselben lokalen Korpus zu ranken.
- **TODO:** Korpus-Loader (download+cache `company_tickers.json`) als Kandidaten-Quelle für `embedMatch`.

## Live-Preis — Stooq als anonymer Fallback (recherchiert)
- **Stooq** (`stooq.com/q/l/?s=nvda.us&e=csv`): **kein Key, anonym**. ABER:
  - **Niedriges Tages-Kontingent pro IP** → „Exceeded the daily hits limit" (exakte Zahl undokumentiert). Mit **Batch + Cache + Breaker** beherrschbar; nicht für 48-Symbol-Bursts.
  - **Nur Symbol, KEINE Klarnamen** (`nvda.us`, nicht „NVIDIA"). → Klarname→Symbol macht der lokale Korpus (oben), DANN Stooq für den Preis.
  - Verzögert/EOD, begrenzte Intraday-Sparkline.
- **Kette:** Resolver → (Yahoo, wenn verfügbar) → **Stooq** → leer. Auflösung immer lokal-fähig.

## News — separate Ebene (für später, recherchiert)
- **Anonym (RSS, wie finanznachrichten):** finanzen.net, wallstreet-online, boerse-online, Der Aktionär, Handelsblatt, ariva, finanztreff; **Google News RSS** (query-basiert); **SEC EDGAR**.
- **Free-Tier mit Key (Opt-in, evtl. Partnerschaft):** Marketaux (5000+ Quellen, Sentiment), Finnhub News (60/min), Benzinga Basic (Headline+Teaser+Link), NewsAPI.org, GNews, NewsData.io, APITube.
- **Bezahlt/gesperrt:** Reuters, Bloomberg — kein freier anonymer API-Zugang (Enterprise/Paywall).
- **SearXNG** (self-hosted, Docker-gated, kein Key): Recherche-Backend für eine optionale Research-KI. Docker-Splash wie skizziert.

## Plan-Notizen des Users (festgehalten)
- **Environmental Triangulation:** mehrere Wires → derselbe Sachverhalt aus 2+ Quellen = höhere Konfidenz. Dedup quellenübergreifend via 3b-News-IDs + Embedding-Seam.
- **News-Overflow:** Per-Quelle-Cap + Embedding-Dedup + Relevanz-Ranking.
- **Lange Artikel → Headlines komprimieren:** Erweiterung der Compose-Stufe (Artikel rein → eine Zeile).
- **Opt-in mit Key, evtl. Partnerschaft** (Benzinga/Marketaux) — später.
