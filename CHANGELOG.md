# Was hat sich geändert?

- Das Fear-&-Greed-Barometer ist jetzt klickbar: Öffnet die CNN-Seite im Browser.
- Schlagzeilen sagen nicht mehr „die Community", wenn sie den Käfig meinen.
- Die Preis-Zuweisung wurde verbessert.
- Die Einstellung „Wiederholungen filtern" wurde entfernt. Der Wiederholungsschutz greift jetzt vollautomatisch.
- Embedding wurde entfernt. Ergo: Die Installation ist rund 600 MB leichter - der alte Ballast wird beim nächsten Update automatisch weggeräumt. Neuere Installer zeigen dafür jetzt einen eigenen Cleanup-Schritt. Ältere führen den Schritt trotzdem aus.
- Der Installer ist jetzt robuster und flackert bei der MB/s Anzeige nicht mehr. Dafür muss der Installer erneut installiert werden, damit der Fix greift. Der alte funktioniert weiterhin wie gewohnt
- MacOS: Die Knöpfe oben rechts in der Titelleiste (Zahnrad, Herz, Update) reagieren jetzt sofort. Vorher hingen sie ein bis zwei Sekunden, weil macOS bei jedem Klick in die Titelleiste erst abwartete, ob daraus ein Fenster-Ziehen wird.
- Cleanup im Maschinenraum: Der Code hinter dem Terminal wurde von Grund auf entwirrt und in saubere, kleine Bausteine zerlegt.
