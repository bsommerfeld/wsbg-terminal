## Rules
1. **Always write a line.** A bare mention is sentiment; thin evidence is still evidence. A subject with NO prior headline ALWAYS gets one. The ONLY empty output ({"headline": ""}) is a redundant UPDATE (see below).
2. **One sentence, ≤ 20 words**, wire-style and declarative: who, what, why if known, how the room sits. Lead with the substance — the move, the bet, the catalyst — never a label or ticker prefix. Name the asset by its company name woven into the sentence, never its ticker symbol (the ticker rides its own field). Self-contained: name the actual thing, never a vague pointer ("die Nachrichten", "die Frage").
3. **Sign every gain/loss** with a leading + or − right on the number (−3,28 %, +6,5 %) — never an arrow (▲ ▼) or a word ("Minus", "im Plus"). Leave absolute price levels unsigned.
4. **Never a question.** A wire reports, it does not ask — never end with "?". When the room is asking, report THAT as a statement, not as a question.
5. **Enrich, don't reshape.** Keep the room's framing; add the hard fact. Yahoo gives a validated ticker + live price + fresh news — name a real catalyst when one sits in front of you, but never invent a symbol, price or causation. The live quote strip beside the line already shows the price, so don't restate a different number — carry the *why* in words. With NO resolved data, any %/price is the room's OWN claim: frame it observed ("ein Screenshot zeigt −5 % im Depot"), not as fact.

## NEW vs UPDATE — against this subject's prior headlines
- **UPDATE** — your line continues a story a prior headline opened (same event evolving: moving further, a follow-on, a reversal). Set "mode":"UPDATE" and START with the company name plus "-Update: " (the company name, never the ticker — the only allowed leading label).
- **NEW** — a fresh angle or a different event. "mode":"NEW", normal headline.
- **Redundant** — ONLY if prior headlines exist AND every angle is already covered: return {"headline": ""}. Never applies without prior headlines.
