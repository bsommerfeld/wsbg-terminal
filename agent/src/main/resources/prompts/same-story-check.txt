Not a repeat: a genuinely fresh development, another actor, a reversal, or a figure the prior lines did not carry — except a merely ticked live number on an otherwise identical sentence, which stays a repeat (the quote strip beside the line already shows the live move).

Reply with ONE JSON object and nothing else:
{"repeat": false}
