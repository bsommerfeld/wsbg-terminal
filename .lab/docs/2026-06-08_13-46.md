# Recherche — Offline-Ticker-Liste + Free-Spark-Provider (Quellen & Links)

Ziel: **Ticker-Auflösung Offline-first** (lokale Liste = erste Anlaufstelle, Yahoo = Fallback). Live-Preise (Spark) bleiben extern (Yahoo/Stooq/anderer Provider). Abdeckung gefordert: **NASDAQ + XETRA + viele Börsen**, auch kleinste Stocks/ETFs.

## A) Symbol-/Ticker-Listen (für Offline-Auflösung Name↔Ticker)
**Downloadbar, kein Key (bevorzugt):**
- **GitHub `LondonMarket/Global-Stock-Symbols`** — JSON/txt/CSV: LSE, NYSE, NASDAQ, Toronto, **Frankfurt/Xetra**, ASX, Tokyo, HK (+ Profile für LSE/NYSE/NASDAQ/Tokyo). → **primär** (Xetra + NASDAQ + global). https://github.com/LondonMarket/Global-Stock-Symbols
- **GitHub `rreichel3/US-Stock-Symbols`** — NASDAQ/NYSE/AMEX, JSON, **per GitHub Actions auto-aktualisiert** → frischestes US. https://github.com/rreichel3/US-Stock-Symbols
- **SEC `company_tickers.json`** — offizielle US-Liste Name↔Ticker, anonym, gepflegt. https://www.sec.gov/files/company_tickers.json
- **dumbstockapi.com** — liest öffentliche Börsen-Files → JSON (NYSE/NASDAQ/JPX…). https://dumbstockapi.com/
- **Stooq DB** — 12.000+ globale Symbole als Download. https://stooq.com/db/

**API mit Free-Tier (Key, am umfassendsten/gepflegt, falls statische Dumps nicht reichen):**
- **EODHD Exchanges/Tickers** — 60+ Börsen inkl. **XETRA**, JSON/CSV. https://eodhd.com/financial-apis/exchanges-api-list-of-tickers-and-trading-hours

**„Wo bei IPOs/Updates suchen" (Primärquellen, autoritativ):**
- **Nasdaq Trader Symbol Directory** (US, tägliche Files): http://www.nasdaqtrader.com/trader.aspx?id=symboldirdefs
- **Deutsche Börse / Xetra Instrument-Liste** (T7 / Cash-Market Reference Data): https://www.xetra.com/xetra-en/instruments/instruments
- Die GitHub-Repos oben (auto-updated) als bequemer aggregierter Spiegel.

**Plan:** Loader lädt + cached eine/mehrere dieser Listen → `EmbeddingService`-Ranker (Tier 2) löst Name→Ticker **offline** auf (erste Anlaufstelle); Yahoo nur noch Fallback/Beschleuniger.

## B) Spark / Intraday-Preise (free)
**Anonym, kein Key:**
- **Stooq** — CSV, anonym, ABER niedriges Tageslimit/IP + meist EOD/verzögert (begrenzte Intraday-Sparkline). https://stooq.com/q/l/
- **Styvio** — kein Key, ~20 min verzögert (Abdeckung/Zuverlässigkeit unbestätigt). https://www.styvio.com/

**Free-Tier mit Key (Opt-in; echtes Intraday):**
- **StockData.org** — 100 req/Tag gratis, Intraday + 7J Historie. https://www.stockdata.org/
- **Finnhub** — Free-Tier 60/min. https://finnhub.io/
- **Alpha Vantage** — Free JSON/CSV, Intraday. https://www.alphavantage.co/
- **Financial Modeling Prep** — 1-Min-Intraday. https://site.financialmodelingprep.com/developer/docs

**Ehrliches Fazit:** Einen **anonymen + freien + umfassenden + Echtzeit**-Spark gibt es nicht. Anonym-Default = **Stooq** (verzögert/limitiert); für echtes Intraday → **Free-Key-Provider als Opt-in** (StockData.org/Finnhub/AlphaVantage/FMP). Architektur: Resolver/Spark-Provider-Interface (wie EmbeddingService-Seam) → Quellen austauschbar, Yahoo nur eine davon.

Sources (Recherche 2026-06-08): GitHub-Repos, SEC, EODHD, dumbstockapi, Stooq, StockData.org, Finnhub, Alpha Vantage, FMP.
