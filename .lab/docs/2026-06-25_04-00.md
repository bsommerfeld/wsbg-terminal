## ✅ ERLEDIGT

### Live-Preise — Multi-Source-EUR-Kette
Neue Module: **`lang-schwarz`**, **`tradegate`**, **`nasdaq-quote`**; Seam in `core`
(`price/PriceSource`, `price/PriceRef`), Orchestrator in `terminal/price/FallbackPriceSource`.
- **L&S Tradecenter** = primäre Echtzeit-EUR-Quelle. Suche per **Name** (`…/instrument/search/main`)
  → instrumentId + ISIN; Chart (`…/chart/dataForInstrument?series=intraday`) → Spark + letzter Preis
  + Vortagskurs (`info.plotlines`). **Namensnormalisierung** war der Schlüssel („Micron Technology, Inc."→„Micron").
- **Fallback-Kette:** L&S → NASDAQ → Yahoo → **Tradegate (allerletzter Notnagel)**.
- **MEZ-Zeitfenster steuern die Primärquelle** (NICHT der unzuverlässige Venue-Timestamp):
  07:30–23:00 → L&S · 23:00–02:00 → NASDAQ (US-Nachbörse) · **02:00–07:30 Lücke = letzter Snapshot, kein Fetch**
  · WE-Slots Sa 10–13 / So 17–19. `FallbackPriceSource.windowAt(ZonedDateTime)`.
- **NASDAQ-Nachbörse korrekt:** `primaryData`/`isRealTime` (NICHT `secondaryData` = alter Schluss) → echter
  $1.214 statt $1.047. Plus `/chart`-Spark (kein `&fromdate`!).
- **EUR-Normalisierung** der USD-Quellen via Live-`EurUsdMonitorService`-Rate.
- **Equity-Guard:** Krypto (-USD) / Index (^) / FX (=X) gehen NIE an L&S/Tradegate (sonst „Bitcoin"→„Bitcoin Group SE").
- **Gap-Fix:** own-Ticker wird im Gap NICHT via Yahoo-Rest-Batch nachgeladen → Unit behält letzten Snapshot
  (`SubjectUnit.updateResolved` überschreibt nicht mehr mit null).
- **`isFresh`-Skew-Fix:** L&S-Zeitstempel liegen ~1 Min in der Zukunft → `Math.abs(age) ≤ FRESH` (sonst gewann L&S nie).

### Browser-Joker erweitert (`CefWebFetcher` / `CefFetchClient`)
